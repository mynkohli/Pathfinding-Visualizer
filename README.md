# Pathfinding Visualizer

An interactive pathfinding visualizer built using Python and Pygame.

## Features

- BFS Pathfinding Algorithm
- Interactive Grid System
- Obstacle/Wall Creation
- Real-Time Visualization
- Start and End Node Selection

## Tech Stack

- Python
- Pygame

## How to Run

Install pygame:

```bash
pip install pygame
```

Run project:

```bash
python main.py
```

## Controls

| Action | Control |
|---|---|
| Select Start Node | Left Click |
| Select End Node | Left Click |
| Create Walls | Left Click |
| Remove Walls | Right Click |
| Run Algorithm | SPACE |
